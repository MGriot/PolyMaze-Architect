# User Guide & Controls

## Main Menu
Navigate using the following keys:
- **G**: Change between **Square** and **Hexagonal** layouts.
- **L**: Set number of **3D Levels** (up to 4 floors).
- **M**: Toggle **Multi-Path** (Braided) mode.
- **R**: Toggle **Trace** (Breadcrumbs).
- **T**: Toggle **Dark/Light** theme.
- **ENTER**: Begin Architecting.

## Exploration (In-Game)
- **Arrow Keys**: Move your avatar.
- **U / D**: Move **Up** or **Down** levels when standing on a Stair (triangular markers).
- **M**: Toggle **Architectural Map** (Vertical exploded view).
- **S**: Toggle **AI Solution** animation.
- **TAB**: Change the AI solver algorithm.
- **P**: **Print** (Save high-res PNG export).
- **ESC**: Back to Menu.

## Visual Indicators
- **Cyan Triangle**: Stair leading Up.
- **Brown Triangle**: Stair leading Down.
- **Crimson Circle**: Goal (Exit).
- **Green Circle**: Your Avatar.
